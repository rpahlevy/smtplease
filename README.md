
# SMTPlease

"Send My Text - Please" or SMTPlease is a small python project that can receive email and retrieve the list of emails using SMTP and IMAP interface. It also provides API interface for both function.

## Project Structure
```
smtplease/
├── app/
│   ├── __init__.py
│   ├── main.py
│   ├── models.py
│   ├── crud.py
│   ├── schemas.py
│   └── database.py
├── Pipfile
├── Pipfile.lock
├── nixpacks.toml
└── README.md

```
